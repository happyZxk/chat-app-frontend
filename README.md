
## 克隆项目

```base
git clone https://github.com/happyZxk/chat-app-server.git
```
## 安装依赖

```bash
$ pnpm install
```

## 运行项目

```bash
# development
$ pnpm  start

# watch mode
$ pnpm  start:dev

# production mode
$ pnpm  start:prod
```

## 项目测试命令

```bash
# unit tests
$ pnpm  test

# e2e tests
$ pnpm  test:e2e

# test coverage
$ pnpm  test:cov
```
